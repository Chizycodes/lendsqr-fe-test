[Node](http://nodejs.org/) is really easy to install & now include [NPM](https://npmjs.org/).

## Install

    $ git clone https://github.com/Chizycodes/lendsqr-fe-test.git
    $ cd lendsqr-fe-test
    $ npm install

## Start & watch

    $ npm start
To start project in development mode 
## Simple build for production

    $ npm run build

## Run tests

    $ npm test
